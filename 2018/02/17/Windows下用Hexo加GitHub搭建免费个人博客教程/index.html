Windows下用Hexo加GitHub搭建免费个人博客教程

作为一个技术型中二少年，有个博客一直是我的梦想

可惜当时的我太过青涩，沉迷金山游侠无法自拔，迟迟没有去钻研

直到有一天我看到了这个东西——Hexo

快速简洁且高效的博客框架

其实当时我是不太喜欢这个东西的
因为它是静态的，干什么都要比动态的WordPress稍微麻烦点
直到一天我发现这个——GitHub Pages
这是个GitHub用户的福利，用户可以在上面托管自己的静态的个人网站
（因为是个人网站，所以一个账号只有一个啦）
最主要的是——无限流量无限时长啊！
有便宜占好事多好啊
于是我决定——搭建自己的个人博客

##准备工作

  • Git客户端
  • Node.js的安装
  • GitHub账号的申请

###Git客户端
因为我是Windows用户，所以我选择了Git For Windows
进入之后点击Download下载安装就好了

###Node.js的安装
进入Node.js官网
点击左面那个最新的LTS版本下载安装就好了

###GitHub账号的申请
首先当然是要进入GitHub
进去之后直接就可以注册

第一列是用户名
第二列是Email
第三列是密码
输完之后点击Sign up for GitHub注册就好了

确认一下选择的是第一个免费计划就可以Continue了

下个页面是要填写你的个人情况，不想填可以skip this step

然后去邮箱验证一下就OK
